# 🛍️ Modern Store Website

A modern and responsive store website built using **HTML**, **CSS**, and **JavaScript**. The project showcases a clean and elegant shopping experience with interactive UI components, responsive layouts, and modern front-end design techniques.

---

## ✨ Features

* 📱 Responsive Navigation Bar
* 🔍 Expandable Search Bar
* 🌙 Dark / Light Mode Toggle
* 🪟 Glassmorphism Product Cards
* 🎥 Full-Screen Background Video
* 🎵 Custom Audio Player
* 📍 Embedded Google Maps Location
* 📱 Fully Responsive Design
* 🎨 Modern Typography
* ⚡ Smooth User Experience

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)
* Google Fonts
* SVG Icons
* Google Maps Embed API
